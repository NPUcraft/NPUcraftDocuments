GSit

SUPER2FH2025年3月21日大约 3 分钟FunctionalPluginLobbySurvival-MainSurvival-IndustrySurvival-Resource

GSit 是一款面向 Minecraft 服务器的坐姿与姿态插件：除了坐在楼梯、半砖等方块上休息外，还支持躺卧、爬行、旋转等地面动作，以及与其他玩家叠坐互动，适合大厅与生存子服里增加社交与角色扮演感。本站侧栏条目名为「Gist」，与早期命名一致，内容即 GSit。

使用前

指令是否开放、冷却与权限以各子服实际配置为准；若提示无权限或指令不存在，请以群内公告或管理说明为准。下列为常见默认用法，便于快速上手。

一、坐下

  • /sit：在当前脚下可站立的方块上原地坐下（需脚下有合适支撑；空中或不可站立位置通常无法使用）。
  • 右键方块：对准部分可交互为「座位」的方块（如 楼梯、半砖、地毯 等，具体以服务器配置为准）的顶面右键，可像坐椅子一样落座，比纯指令更符合直觉。
  • 起身：多数情况下 按住潜行（Shift） 即可结束坐姿；若无效，可小幅移动或再按一次相关指令（以当前版本为准）。

二、躺卧与扑倒

  • /lay：在地面舒展躺下，适合挂机展示或截图。
  • /bellyflop：肚皮朝下扑倒，偏趣味动作。

起身方式通常同样为 潜行，或再次输入指令结束状态。

三、爬行与旋转

  • /crawl：进入爬行姿态，贴地移动，适合钻低矮缝隙或趣味玩法。
  • /spin：原地旋转（「爱的魔力转圈圈」），多用于娱乐与互动。

在狭窄空间或贴墙时可能出现模型与方块轻微穿插，属于常见现象；若长时间卡住，可尝试起身、走动或重进子服。

四、与其他玩家互动（叠叠乐）

  • 在服务器允许的前提下，可对其他玩家使用右键（具体是否需空手、是否需潜行，以服内规则与插件配置为准），坐到对方头上或身上，多人依次叠高形成「人塔」。
  • 请尊重他人体验：勿在对方明确表示拒绝时强行叠坐；部分活动区或 PVP 场景可能限制此类行为，以管理组规定为准。

五、指令速查

/sit          # 原地坐下
/lay          # 躺下
/bellyflop    # 肚皮扑
/crawl        # 爬行
/spin         # 旋转

部分整合包或子服会为上述功能设置别名或冷却时间；若与上表不一致，以游戏内 Tab 补全与管理说明为准。

六、常见问题

  • 坐下后动不了或视角异常
    先 潜行 起身；仍异常可尝试切换第三人称视角（F5）或重新进入该子服。

  • 右键方块无法坐
    可能该方块未在服务器配置为可坐类型，或当前手持物品阻止交互；可换 空手 再试，或改用 /sit。

  • 想查更多子命令或权限名
    请查阅上方 Modrinth / Spigot 说明，或在有权限时使用服内帮助类指令（若管理员已开启）。

与模组、资源包

纯原版 + 插件环境下 GSit 一般可正常使用。若同时使用 大型模组 或 自定义玩家模型，偶发姿势错位时可尝试关闭冲突模组或向管理员反馈。
