玩家商店由 **QuickShop-Hikari** 驱动，成交使用与经济系统一致的 **西瓜币（🍉）**，余额与扣款方式见 [经济系统总览](./README.md) 及 [CoinsEngine 常用命令](./README.md#coinsengine-player-commands)。本文整理**服内常用操作**；面板文案、点击方式等可能随版本或管理组配置微调，完整机制请以官方文档为准。

::: info 完整文档

[QuickShop-Hikari 功能文档（中文）](https://quickshop-community.github.io/QuickShop-Hikari-Documents/zh-CN/docs/category/features)

:::

::: tip 适用范围

本站文档标签以**主服**为主；其他生存子世界是否允许设店、有无额外领地要求，以游戏内公告与管理组说明为准。设店位置建议在已圈 [领地](./Residence.md) 内，避免容器被他人破坏。

:::

## **一、创建出售商店**

基本流程：**放置容器 → 用「要卖的物品」指向商店 → 左键 → 在聊天栏输入价格**。

| 操作步骤 | 结果 |
|:--|:--|
| 放置容器（如箱子、木桶等插件允许的容器） | 作为商店库存方块 |
| **手持与将来售卖相同的物品**，在**生存模式**下**左键**该容器 | 将该物品登记为商店货种（名称、附魔、模型数据等是否严格匹配由服务端 ItemMatcher 配置决定） |
| 在聊天栏**输入价格数字**并发送 | 创建成功（例：输入 `10` → 单价 **🍉**） |

::: warning 公开售卖与左键购买

商店创建后，在满足「未冻结、有货、对方余额充足」等条件时，**他人通常可通过左键购买**。请确认定价、位置与库存；必要时先**冻结**再改价或补货，并用 [领地](./Residence.md) 保护容器，降低误触与恶意扫货风险。

:::

## **二、顾客购买**

- **左键**商店（具体交互若与本文不一致，以游戏内提示为准）发起购买，系统扣除 **🍉** 并将物品放入背包。
- 购买前可通过 **`Tab`**、`/balance` 等确认余额；背包**空间不足**可能导致交易失败或部分失败，请预留空位。
- 若购买无反应：可能是**售罄**、商店**冻结**、**无经济权限**或处于无法交互的区域，可换店或联系店主。

## **三、补充库存**

- 将**与商店货种一致**的物品放入商店绑定的容器，**可售数量**会随库存增加而更新。
- **出售模式**下：你作为店主向外卖货，补的是「待售出」的物品。
- **收购模式**下：向卖家支付 **🍉**、收纳货物等由插件与容器联动完成；补货、取钱、取货等请严格按游戏内面板提示操作（不同版本文案可能不同）。

## **四、切换收购模式**

- **右击**商店信息牌（或商店方块，以服内交互为准）打开控制面板。
- 找到 **`商店模式: 出售 [更改]`** 一类选项并切换，将商店改为**收购模式**。
- **收购模式**下，其他玩家可把指定物品卖给你并获得 **🍉**（需你账户与商店配置允许）；与出售模式互补，适合收原材料、生物掉落物等。

## **五、冻结商店**

- **右击**打开控制面板，找到 **`冻结模式: ✘ [切换]`**（或类似字样）并开启。
- 冻结后**一般玩家无法完成购买/出售**，用于维护、改价、临时歇业或防止误扫。
- 恢复营业时在同一位置**关闭冻结**即可。

## **六、清空库存**

- 直接从商店容器中**取出**物品，可售数量会相应减少；取空即相当于无货状态，顾客将无法购买（直至再次补货）。

## **七、命名商店**

对准商店执行：

```bash
/quickshop name <名称>
```

名称会出现在日志、部分提示或交易记录中，便于区分多店；主命令在部分环境下也可能是 `/qs` 等别名，以 **`/quickshop help`** 或服内说明为准。

## **八、修改价格**

- 打开商店控制面板（通常**右击**信息牌或商店方块）。
- 点击 **`价格: xx [更改]`**，按提示在**聊天栏**输入新价格数字。
- 建议在**冻结**或低峰时段改价，减少玩家在你输入价格瞬间以旧价成交的争议。

## **九、移除商店**

- **方法一**：在确认无纠纷后**破坏**商店容器（注意领地与掉落物；具体是否一并移除商店以插件行为为准）。
- **方法二**：在控制面板中寻找**删除 / 移除商店**类选项（文案因版本而异），按提示确认。

## **十、物品预览**

### 1. 基础预览

- 点击商店打开交易界面，将鼠标悬停在 **`[预览物品]`**（或同类按钮）上查看简介。

### 2. 高级预览

- 点击 **`[预览物品]`** 打开完整展示界面，便于核对附魔、耐久、命名等是否与预期一致（尤其购买高价或「唯一」物品时）。

## **常见问题**

| 现象 | 可能原因与处理 |
|:--|:--|
| 无法创建商店 | 无权限、该位置禁止建店、容器类型不支持、未手持正确物品；查看聊天栏报错或询问管理组。 |
| 左键无反应 | 非生存模式、手持物品与已登记货种不一致、商店已冻结、跨领地无交互权限等。 |
| 提示余额不足 | 使用 `/pay` 收款前确认对方未关收款；自己购物前先 [查看余额](./README.md#coinsengine-player-commands)。 |
| 收购不生效 | 未切换到收购模式、定价/单次交易量不合理、店主余额不足以支付等，检查面板与经济插件记录。 |
