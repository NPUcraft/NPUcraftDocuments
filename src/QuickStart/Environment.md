入服前请在本机准备好 **Java 运行环境** 与 **支持外置登录的启动器**，并完成皮肤站账号相关设置（详见 [账号注册](./Register)）。本节说明推荐版本、安装与验证方式，以及常见踩坑。

::: tip 服务器当前版本

Java 版客户端版本以 **[更新日志](/Overview/Changelog)** 为准；请尽量与文档中「服务器当前版本」一致或在其兼容范围内，以减少进服失败或材质/协议问题。

:::

## **一、Java 运行环境**

NPUcraft 使用较新的 Minecraft Java 版服务端，**建议在本机安装 Java 21（LTS）**，与当前周目版本相匹配。

### **1. 获取与安装**

1. 从 [Oracle JDK 下载页](https://www.oracle.com/java/technologies/downloads/) 选择 **Java 21**，并下载与你的操作系统、架构（Windows x64、Apple Silicon / Intel 等）对应的安装包。
2. 运行安装程序，**一路默认路径即可**；若安装程序询问是否设置 `PATH`，请保持勾选。
3. 若你更倾向使用 **OpenJDK 发行版**，可选用 [Eclipse Temurin](https://adoptium.net/) 或 [Microsoft Build of OpenJDK](https://learn.microsoft.com/java/openjdk/download) 等，同样选择 **21** 系列。

### **2. 验证是否安装成功**

- **Windows**：`Win + R` → 输入 `cmd` → 回车，执行：
  ```bash
  java -version
  ```
  输出中应出现 `21.x.x`（或厂商字符串中包含 21）。
- **macOS / Linux**：打开终端，同样执行 `java -version`。

::: note 多版本 Java 并存

若系统中曾安装过 Java 8/17 等，请确认启动游戏时所使用的仍是 **Java 21**（多数启动器可在「全局设置 / Java 路径」中指定）。版本过低可能导致无法启动高版本客户端。

:::

## **二、Minecraft 启动器与外置登录**

你需要使用 **支持外置登录（Authlib Injector / 类似机制）** 的启动器，以便使用 [NPUskin 皮肤站](https://skin.npucraft.com) 账号进入服务器。本文以 **[PCL2](https://afdian.com/p/0164034c016c11ebafcb52540025c377)** 为例；使用 **HMCL** 等启动器时，填写项名称可能略有不同，但 **认证服务器地址** 一致即可。

在启动器中进入 **版本设置 → 对应版本 → 设置 → 服务器 / 认证**（不同启动器菜单位置可能为「账户」「登录」等），填写：

| 描述 | 内容 |
| :--: | :--: |
| 登录方式（**必选**） | 第三方登录：`Authlib Injector` 或兼容项（如 LittleSkin 类） |
| 认证服务器（**必填**） | `https://skin.npucraft.com/api/yggdrasil` |
| 注册链接（可选） | `https://skin.npucraft.com/auth/register` |
| 服务器名称（可选） | `NPUskin皮肤站` |

完成上述配置后，使用皮肤站邮箱与密码在启动器中登录；若尚未注册，请按 [账号注册](./Register) 操作。

::: tip 整合包

社区可能在 QQ **群文件**中提供已配置好的整合包（入口见 [主页](/)「关于我们」中的加群链接）。使用整合包时仍需确认其中的认证地址与上表一致，并注意整合包发布日期是否过旧。

:::

## **三、游戏版本与分配内存**

- **客户端版本**：安装与 **[更新日志](/Overview/Changelog)** 中「服务器当前版本」一致的 **正式版**（除非公告另有说明）。不要在未确认的情况下强行使用快照版进服。
- **分配内存**：在启动器中为该版本分配 **不少于 4 GB** 的堆内存（机器内存紧张时可酌情降低，但过低可能导致卡顿或无法加载资源包）。具体项名称多为「游戏内存」「JVM 参数 `-Xmx`」等。

## **五、常见问题**

**问：`java -version` 提示不是内部命令或找不到？**  
答：可能未安装成功或未加入 `PATH`。可重新安装 JDK 21 并勾选「设置 PATH」，或在启动器中手动指定 `java.exe` 路径。

**问：启动器提示认证失败？**  
答：核对认证服务器是否为 `https://skin.npucraft.com/api/yggdrasil`、账号是否在皮肤站完成验证；密码错误与邮箱未验证是常见原因。仍无法解决时，请携带**完整报错原文**按 [提问的智慧](./HowToAsk) 在群内求助。

**问：能使用正版 Mojang / 微软账号直接进服吗？**  
答：本服务器采用皮肤站外置登录体系，请按本站 [账号注册](./Register) 使用 **教育邮箱或审批渠道** 注册皮肤站账号，并在启动器中走外置登录，而不是仅用正版登录替代认证服务器配置（除非公告另有说明）。

**问：进服后卡顿、掉线？**  
答：先调低视距与画质、关闭不必要光影；仍频繁掉线时记录大致时间与现象，再向管理员或群内反馈。
